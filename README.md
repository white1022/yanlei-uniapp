## 说明

* css部分使用了uni.scss部分变量, 其他语法已剔除特殊语法, 除变量外可直接替换其他css预处理器
* app.vue下定义了个别全局样式以及引用iconfont图标
* 代码注释比较详细，单独使用请主意1.2两条的引入
* 该项目为前端页面效果展示, 数据和图片来自网络，均只用于案例展示，请勿用于其他用途。
